🛍 Mercari 在庫同期ブックマークレット

Mercari の出品ページでワンクリックして、在庫管理システムに自動同期

⚙️ 設定

Inventory Manager のサーバーアドレスを入力してください。

📌 Docker コンテナ内では 8080、リバースプロキシ経由では実際のURL（例: http://192.168.1.203）

📋 使い方

  1. Mercari にログインして、持ち物一覧ページを開きます。
  2. 下のブックマークリンクをマウスでドラッグして、ブラウザのブックマークバーにドロップします。
    またはリンクをクリック → Ctrl+D でブックマーク保存 → 名前「Mercari同期」、URL欄に自動入力されるのを確認。
  3. 持ち物一覧ページでブックマークをクリックすると、自動的に商品データを抽出して Inventory Manager に送信します。
  4. 画面上に進捗表示がオーバーレイで表示され、完了すると「XX件同期完了」と表示されます。

🔖 ブックマークレット（ドラッグしてブックマークバーに）

下のリンクをマウスで掴んで、ブラウザのブックマークバーにドロップしてください。

Mercari 在庫同期

💡 ヒントと注意点

  • Mercari と Inventory Manager が同じネットワーク（同じWi-Fi）に接続されている必要があります。
  • CORSはサーバー側で許可済みです。ブックマークバーから直接実行してください。
  • 商品数が100件を超える場合、自動スクロールで全商品を取得します（数秒〜数十秒かかります）。
  • 既に在庫にある商品は更新され、新しい商品は追加されます。
  • スマホではブックマークレットが使えない場合があります。その場合はPCから操作してください。
