🔄 Mercari 持ち物同期

Mercari の「持ち物一覧」ページから商品データを取得して在庫に同期します。

操作方法
Mercari 持ち物同期
1. Chrome で 持ち物一覧ページ を開く
⚠️ 「持ち物一覧」ページが必要です。「持ち物詳細」や他のページでは動作しません。
2. 下の「同期開始」ボタン領域からスクリプトをコピー
3. F12 → Console に貼り付けて実行
4. 自動で mercari_owned_items.json がダウンロードされ、API に送信されます
5. 「📂 JSON ファイルをアップロード」で選択して完了
💡 ページの URL が /mypage/inventory で始まることを確認してください。SPA ルーティングにより、他の URL では商品データが取得できない場合があります。
