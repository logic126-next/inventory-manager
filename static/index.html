🔖 ブックマークレット

Mercari のブラウザでワンクリック！Cookie 不要、Playwright 不要。

🔖 Mercari在庫同期（ブックマークバーにドラッグ）
使い方：
  1. Mercari にログインし、持ち物一覧ページを開く
  2. 上のリンクをブックマークバーにドラッグ＆ドロップ
  3. 同じ Mercari 持ち物ページで、ブックマークをクリック
  4. 自動的に商品データを抽出・送信 → 「XX件同期完了」と表示される
📖 詳細ページ

💡 サーバーアドレス変更が必要な場合は、/static/mercari-bookmarklet.html にアクセスして設定してください。

🔑 ブックマークレット API キー

API キーを設定すると、ブックマークレットからの同期に認証が必要になります。未設定の場合は SSO 認証のみ。

🍪 Mercari Cookie 設定

Mercari 持ち物・購入履歴の同期にログイン Cookie が必要です。
取得方法：Mercari にログイン後、ブラウザ DevTools → Application → Cookies → _session_id の値をコピー。
または 持ち物ページ の DevTools Console で document.cookie を実行して結果をコピー。

🖼️ 画像をDBにキャッシュ

Mercari CDN の画像 URL を Base64 に変換して DB に保存します。Mercari 側の画像が消えても表示されなくなります。
同期時の新商品も自動的に Base64 で保存されます。

🛒 Mercari 購入履歴同期

購入履歴ページ の商品を一括でインポートします。
ブックマークレット方式：Mercari 購入履歴ページでブックマークをクリックするだけで完了します。
